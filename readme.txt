1. 실행 방법
- execute.bat을 더블클릭하여 실행합니다.
- 실행을 위해서는 PC에 Python이 설치되어 있어야 하며, 환경 변수에 Python 폴더가 지정되어 있어야 합니다. Python을 설치할 때 "PATH에 추가"하는 옵션을 활성화하여 설치해주세요. 
- Python 설치시 PATH 추가 옵션을 활성화하지 않은 경우, 시스템→고급 시스템 설정→환경 변수→사용자 변수의 Path에 Python이 설치된 폴더 경로를 추가해주세요. 명령 프롬프트에서 Python을 입력하였을때 실행되면 됩니다.
- 이 프로그램은 PyQt5, pandas, numpy, win32com, statsmodels 패키지를 사용합니다. 패키지가 설치되어 있지 않은 경우 자동으로 설치하지만, 온라인 연결이 필요합니다. 또한 온라인 연결이 되어있더라도 PC에 따라 설치 오류가 발생할 수 있습니다. 이 경우 기본 Python 엔진을 Anaconda3의 엔진으로 지정하면 문제없이 실행할 수 있습니다.


2. 사용법
- 본 프로그램은 4개의 모듈을 포함합니다. 

1) Rawdata Updater
- 실거래 Rawdata를 최신화하는 모듈입니다.
- 기존 DB 파일을 열어서 import하고 새롭게 받은 DB를 열어서 import합니다. 그후 Append를 눌러서 합치고 저장합니다.
- 기존 DB와 새로운 DB의 칼럼명이 다를 경우 에러가 발생합니다.

2) Data Processor
- Rawdata를 분석용으로 가공하는 모듈입니다.
- 가공을 위해 필요한 데이터는 다음과 같습니다.
  1) Rawdata : 정제되지 않은 실거래 데이터
  2) Key : 주택단지명을 정제한 클리어링용 데이터
  3) Dongcode : 읍면동코드
  4) CPI : 월별 소비자물가지수
- Rawdata의 타입을 지정하고 분석을 진행한 다음, 데이터 가공이 마무리되면 저장합니다.
- 2018년 8월 25일 현재 아파트 매매, 아파트 전세만 지원합니다.
- 필터링 조건
  1) 아파트 매매 : 10평 이상, 서울시
  2) 아파트 전세 : 10평 이상, 전세, 서울시

3) PivotTable Maker
- 실거래 데이터를 통해 기초통계 확인용 피벗 테이블을 제작합니다.
- Rows에는 피벗테이블의 행, Columns에는 열, Subject에는 분석 대상 칼럼을 설정합니다.

4) Regression
- 회귀분석을 통해 인덱스를 계산합니다.
- 가공된 데이터를 불러오면 칼럼명이 표시됩니다. 
- 인덱스 계산을 위해서 Dep Variable에 평당가 칼럼, Continuous Indep Variable에 Floor, Area 칼럼, Time Dummy에 거래년월 칼럼, APT Dummy에 APT명 Key 칼럼을 설정합니다. Location Column에는 분석 목적에 따라 시도/시군구/읍면동 칼럼을 설정합니다. (시도 단위 분석에는 시간이 많이 소요되거나 메모리 에러가 발생할 수 있음)
- OLS 또는 Quantile Regression중 하나를 선택하여 Analyze를 누릅니다.
- 분석이 마무리되면 인덱스 계산결과를 저장합니다.
